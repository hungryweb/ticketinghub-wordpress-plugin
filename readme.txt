Simply put, a booking widget is a widget you can add to your travel website.

== Description ==

We help tour operators increase their ROI by providing a frictionless ticket-buying experience for their customers and aggregating sales from any channel into an intuitive dashboard.

== Installation ==

Installing "TicketingHub Widget" can be done either by searching for "TicketingHub Widget" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
